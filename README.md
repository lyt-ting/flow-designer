# 🚀 Flow-Designer - 流程圖產生器



這是為 線性 SOP 設計的視覺化工具，支援一鍵生成現代化、玻璃擬態風格的流程圖卡片。

## ✨ 核心特色
* **🎨 現代化視覺**：玻璃擬態 (Glassmorphism) 設計。
* **🛠️ 高度自定義**：支援橫向/縱向切換、字體大小精細調整。
* **🔍 智慧圖示**：整合 Lucide Icons，支援手動輸入名稱自動帶出。
* **📸 圖片匯出**：一鍵產出高品質 PNG 圖檔。

## 🚀 快速啟動
如果你想在本地修改這個專案：
1. **安裝依賴**：`npm install`
2. **啟動預覽**：`npm run dev`
3. **發布更新**：`npm run deploy`

## 🛠️ 技術棧
* **Framework**: React 19 + Vite
* **Styling**: Tailwind CSS 4
* **Icons**: Lucide React
